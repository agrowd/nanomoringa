# Branding - Medicina Natural

## Instrucciones

Por favor, agrega en esta carpeta:

### 1. Información de la Empresa
- `empresa-info.txt` - Toda la información de la empresa, misión, visión, productos
- `redes-sociales.txt` - Links de redes sociales, contactos

### 2. Diseño Visual
- `colores.txt` - Paleta de colores de la marca
- `logo/` - Logo en diferentes formatos (SVG, PNG)
- `tipografia.txt` - Fuentes a utilizar

### 3. Contenido
- `productos/` - Información y fotos de productos CBD
- `videos/` - Videos de productos (o links a videos)
- `imagenes/` - Fotos generales, banners, etc.

### 4. Textos
- `textos-landing.txt` - Textos cortos para la landing (importante: breves para Meta Ads)
- `descripciones-productos.txt` - Descripciones de productos con CBD
